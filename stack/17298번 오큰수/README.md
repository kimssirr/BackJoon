# 오큰수

### 요구사항
오른쪽을 바라볼때 자신보다 큰 원소 중 가장 왼쪽에 있는 원소를 체크해야한다.

### 풀이
stack을 순회하며 현재의 탑과 값을 비교하면서 같거나 작으면 stack에서 뺀다.
현재 원소보다 같거나 작은 수는 앞으로 의미가 없기 때문이다. (현재 원소가 더 까가움 오큰수기때문에!)
stack이 비면 (= 오큰수가 없음) -1을 저장한다.
stack이 비지 않았다면 자신보다 큰 원소가 있기에 가장 가까운 (stack.peek())을 저장한다.
현재 원소를 stack에 저장하고 반복

### 시간복잡도
O(n)
