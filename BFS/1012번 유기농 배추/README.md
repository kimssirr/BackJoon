# 유기농 배추

### 요구사항
- 배추가 상하좌우로 연결된 묶음의 개수를 구하는 문제.
- map은 처음에 전부 0으로 생성하고, 배추가 있는 위치만 1로 설정한다.
- 각 묶음마다 지렁이 1마리가 필요하므로, **묶음의 개수 = 결과값**이다.

### 풀이
1. 입력으로 받은 n × m 크기만큼 map을 0으로 초기화한다.
2. 배추 좌표 (x, y)가 주어질 때마다 map[x][y] = 1로 저장한다.  
   → 0으로만 채워진 map에 배추 위치만 1 넣기
3. 이후 map 전체를 순회한다.
    - `map[i][j] == 1` 이고 `visited[i][j] == false` 라면  
      아직 탐색되지 않은 새로운 배추 묶음을 의미한다.
4. BFS를 실행하여 해당 묶음에 포함된 인접 배추를 모두 탐색한다.
    - 시작 좌표를 queue에 담고 visited 처리
    - 상하좌우로 이동하며 다음 조건을 모두 만족하면 확장
        - 범위 안쪽
        - 아직 방문하지 않음
        - map 값이 1 (배추 존재)
    - map에 i,j 주변에 1 있는지 확인 → 유효성 검사 통과 시 방문 처리 과정
5. BFS가 끝나면 해당 묶음에 대한 탐색이 완료된 것이므로, count(지렁이 수)를 1 증가시킨다.
6. 전체 map을 모두 순회한 후 count를 출력한다.

### 시간복잡도
- BFS는 각 칸을 최대 1번씩 방문 → O(n × m)
- 전체 테스트 케이스 T에 대해 → O(T × n × m)
