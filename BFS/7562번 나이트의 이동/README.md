# 나이트의 이동

### 요구사항
- 체스판 위에서 나이트가 이동할 수 있는 최소 이동 횟수를 구하는 문제.
- 나이트는 총 8방향으로 이동하며, 각 테스트 케이스마다 **시작 좌표 → 목표 좌표까지의 최소 이동 횟수**를 구해야 한다.

### 풀이
1. 체스판 한 변의 길이 len을 입력받는다.
2. visited[len][len] 배열을 false로 초기화한다.
3. 나이트의 위치 이동을 나타내는 8가지 dx, dy 방향을 미리 정의한다.
4. 시작 좌표(startX, startY)를 BFS의 시작점으로 queue에 넣는다.
    - Point(x, y, n)에서 n은 현재까지의 이동 횟수.
5. BFS 진행
    - queue에서 하나씩 꺼내 현재 좌표(now)를 확인
    - 도착 지점(endX, endY)에 도달하면 now.n이 최소 이동 횟수이므로 반환
    - 8가지 방향(nx, ny)에 대해
        - 범위 내이고
        - 방문하지 않았다면  
          → visited 체크 후 queue에 넣기  
          → 이동 횟수(now.n + 1) 전달
6. 도착 지점에 도달하면 BFS 종료 후 결과 출력.

### 시간복잡도
- len × len 체스판의 모든 칸을 최대 한 번씩 방문 → O(len^2)
- 테스트 케이스를 T라 하면 총 O(T × len^2)
