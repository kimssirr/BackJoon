# 빙산

### 요구사항
- N × M 격자에서 매년 빙산이 **주변 바다(0)의 개수만큼 녹는다**.
- 빙산이 두 덩어리 이상으로 분리되는 최초의 해(year)를 출력한다.
- 끝까지 분리되지 않고 **전부 녹으면 0을 출력**한다.
- 연결 여부는 **상하좌우 기준**으로 판단한다.

### 포인트
- 빙하가 녹을때 동시에 녹아야한다. 하나씩 하면 방금 다 녹은 빙하가 바다로 인식되어 잘못 녹는 상황이 발생할 수 있다.

### 풀이

1. 전체 로직은 **while 반복으로 해마다 시뮬레이션**한다.

2. `check()`
    - 현재 격자를 한 번 스캔하며 빙산 칸 개수(`len`)를 센다.
    - 빙산이 하나도 없으면 `-1` 반환.
    - 빙산이 있으면, 임의의 한 빙산 칸에서 BFS를 시작해  
      **모든 빙산이 하나로 연결돼 있는지** 확인한다.
    - BFS로 방문한 빙산 수가 `len`과 다르면 → 이미 분리된 상태.

3. `check()` 결과 처리
    - `-1` → 빙산이 전부 녹음 → `0` 출력 후 종료
    - `1` → 빙산이 분리됨 → 현재 `year` 출력 후 종료
    - `0` → 아직 한 덩어리 → 계속 진행

4. `minus()`
    - 모든 빙산 칸에 대해 주변 바다(0) 개수를 계산한다.
    - 감소량을 큐에 저장한 뒤 한 번에 반영해  
      **모든 빙산이 동시에 녹도록 처리**한다.

5. `year++` 후 다음 해로 넘어가며 반복한다.

### 시간복잡도
- 한 해마다
    - 전체 스캔 + BFS + 녹이기 모두 **O(N × M)**
- 총 시간복잡도: **O(Y × N × M)**  
  (Y = 분리되거나 전부 녹을 때까지의 년 수)
- 공간복잡도: **O(N × M)**
