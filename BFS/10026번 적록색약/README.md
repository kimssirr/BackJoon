# 적록색약

### 요구사항
- 일반인의 시야에서는 ‘R’, ‘G’, ‘B’가 모두 다른 색으로 보인다.
- 적록색약 시야에서는 ‘R’과 ‘G’를 같은 색으로 본다.
- 두 상황에서 각각 몇 개의 구역(연속된 동일색 영역)이 있는지 구하는 문제.
- 즉, **정상 시야 구역 개수** + **적록색약 시야 기준 구역 개수** 두 값을 모두 출력해야 한다.

### 풀이
1. 입력받은 n × n 격자를 char[][] map에 저장한다.
2. **정상 시야용 BFS**
    - `map[x][y]` 와 동일한 색만 이어갈 수 있다.
    - 방문하지 않은 칸을 만나면 BFS 시작 → 구역 1개 증가.
    - isValid()에서 다음 조건 체크
        - 범위 안쪽
        - 방문하지 않음
        - map[x][y] == map[nx][ny]  (완전히 같은 색)
3. 정상 시야 구역 개수를 센 뒤, visited 배열을 다시 초기화한다.
4. **적록색약 시야 BFS**
    - ‘R’과 ‘G’는 같은 색으로 취급한다.
    - isValid2()에서 다음을 판별
        - 현재 색이 R/G이고, 다음 색도 R/G이면 연결
        - 현재 색이 B면, 다음도 B일 때만 연결
    - 역시 방문하지 않은 칸을 만나면 BFS2 시작 → 구역 1개 증가.
5. 최종적으로 정상 시야 구역 수 count, 적록색약 시야 구역 수 count2를 출력한다.

### 시간복잡도
- 전체 grid를 BFS로 2번 탐색
- BFS는 각 칸을 최대 1번 방문하므로 → O(n^2)
