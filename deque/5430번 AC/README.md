# AC

### 요구사항
배열을 실제로 뒤집지 않고,
뒤집기(R)와 삭제(D)를 효율적으로 수행해야 한다.
R이 나올 때마다 방향만 반전시키고 deque에서 pop 방향만 바꾼다.

### 풀이
1. 입력 배열을 ArrayDeque에 저장한다.
2. flag 변수를 두어 현재 방향을 관리한다.
false: 정방향 / true: 역방향
3. 명령 문자 순회
R: flag 반전
D: flag에 따라 pollFirst() 또는 pollLast() 수행, 비어 있으면 error

4. 출력
flag에 따라 정방향이면 앞에서부터, 역방향이면 뒤에서부터
요소를 꺼내어 결과 문자열을 만든다.

### 시간복잡도
명령 문자열 길이를 n이라고 하면
→ 탐색 O(n)
deque 출력 O(n)

총 O(n)
