# 좋은 단어


### 요구사항
- 문자열에서 인접한 동일 문자 두 개가 나오면 서로 소거한다.
- 모든 문자가 소거되어 스택이 비면 '좋은 단어'로 인정한다.
- N개의 단어 중 좋은 단어의 개수를 구하는 문제.

### 풀이
1. 단어를 한 글자씩 순회한다.
2. ArrayDeque를 스택처럼 사용한다.
    - 스택이 비어있지 않고 현재 문자 == 스택 top → pollLast()로 소거
    - 그렇지 않으면 addLast()로 push
3. 순회가 끝난 뒤 스택이 비어 있으면 좋은 단어 → result++
4. 모든 단어 검사 후 result 출력한다.

### 시간복잡도
- 단어 길이를 n이라 하면 각 글자가 push/pop 1회 → O(n)
- 단어 수를 N개라 하면 전체 → O(N·n)
