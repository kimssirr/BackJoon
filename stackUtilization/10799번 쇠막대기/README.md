# 쇠막대기

### 요구사항
- 레이저는 `()` 형태이며, 레이저 위치에서 현재 스택의 쇠막대기 개수만큼 조각이 생긴다.
- 쇠막대기의 끝 `)` 는 막대 하나가 끝났음을 의미하므로 조각 1개가 추가된다.
- 입력된 괄호 문자열을 기반으로 총 조각 수를 구하는 문제.

### 풀이
1. 문자열을 앞에서부터 순회한다.
2. `(` 를 만나면 스택에 push 한다.
3. `)` 를 만나면 두 가지로 나뉜다.
    - 이전 문자가 `(` 인 경우 → 레이저
        - 스택에서 `(` 제거
        - 현재 스택 크기만큼 조각 추가
    - 이전 문자가 `)` 인 경우 → 쇠막대기 끝
        - 스택 pop
        - 조각 1개 추가
4. 전체 문자열을 순회한 후 result 출력.

### 시간복잡도
- 문자열 길이를 n이라 하면  
  push/pop 각각 O(1)  
  전체 순회 O(n)
